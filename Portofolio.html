Proyek Saya

Prediksi Harga Komoditi Pertanian Jagung

Proyek ini adalah bagian dari Proyek akhir 1 yang mengembangkan Toke Jagung, sebuah platfrom berbasis web untuk memberikan informasi harga jagung secara akurat dan memudahkan transaksi jual-beli jagung secara online. Website ini menampilkan:

  • Harga jagung harian, mingguan, dan tahunan di wilayah Balige, Porsea, dan Sitoluama.
  • Prediksi harga jagung menggunakan data historis untuk membantu petani dan pedagang dalam mengambil keputusan.
  • Fitur penjualan langsung dari pengguna kepada Tokke untuk kemudahan transaksi.

Teknologi: HTML, CSS, JavaScript, PHP, MySQL

Sistem Pemberi Pakan Ayam Otomatis Berbasis IoT dan Blynk

Proyek ini merupakan tugas Proyek Akhir 2 yang bertujuan untuk merancang sistem pemberi pakan ayam otomatis berbasis Internet of Things (IoT). Sistem ini menggunakan aplikasi Blynk untuk mengontrol pemberian pakan secara otomatis maupun manual melalui smartphone, serta memonitor kondisi pakan dan air untuk mendukung efisiensi peternakan modern.

Teknologi: Arduino, ESP32/ESP8266, Blynk, Sensor Ultrasonik, Relay

Sistem Pengering Tangan Otomatis

Proyek ini merupakan tugas mata kuliah Sistem Tertanam yang dirancang untuk membuat alat pengering tangan otomatis. Sistem bekerja dengan sensor ultrasonik yang mendeteksi keberadaan tangan dan mengaktifkan kipas DC melalui relay yang dikendalikan oleh Arduino Mega.

Teknologi: Arduino Mega, Sensor Ultrasonik, Relay, Fan DC

Sistem Pendeteksi Ketinggian Air Danau Toba Berbasis LoRa

Proyek ini merupakan Tugas Akhir yang dirancang untuk memantau ketinggian air Danau Toba secara real-time menggunakan teknologi Internet of Things (IoT). Sistem ini memanfaatkan sensor JSN-SRT04 untuk mengukur jarak permukaan air dan anemometer untuk memantau kecepatan angin di sekitar danau. Data dikirim secara nirkabel melalui modul LoRa dan dapat dipantau dari jarak jauh melalui sistem monitoring berbasis web.

Teknologi: LoRa, ESP32, Sensor JSN-SRT04, Anemometer, Panel Surya Sistem IoT
